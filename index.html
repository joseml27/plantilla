Tipos de memoria Ram

Memoria Ram

La memoria de acceso aleatorio (Random Access Memory, RAM) se utiliza como memoria de trabajo de computadoras y otros dispositivos para el sistema operativo, los programas y la mayor parte del software. En la RAM se cargan todas las instrucciones que ejecuta la unidad central de procesamiento (CPU) y otras unidades del computador, además de contener los datos que manipulan los distintos programas.

Static RAM (SRAM) .

SRAM (RAM estática) es una memoria de acceso aleatorio (RAM) que retiene bits de datos en su memoria mientras se suministra energía . A diferencia de la RAM dinámica (DRAM), que almacena bits en celdas que constan de un condensador y un transistor, la SRAM no tiene que actualizarse periódicamente.

Dynamic RAM (DRAM)

DRAM, pronunciada DEE-RAM, se utiliza ampliamente como memoria principal de una computadora . Cada celda de memoria DRAM está compuesta por un transistor y un capacitor dentro de un circuito integrado, y un bit de datos se almacena en el capacitor.

Single Data Rate Synchronous Dynamic RAM (SDR SDRAM)

Las memorias SDR SDRAM son memorias síncronas, con tiempos de acceso de entre 25 y 10 ns y que se presentan en módulos DIMM de 168 contactos. ... El nombre correcto es SDR SDRAM ya que ambas (tanto la SDR como la DDR) son memorias síncronas dinámicas. El tamaño máximo de memoria en este tipo de RAM es de 512mb.
